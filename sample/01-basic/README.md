# 01-basic：基本サンプル

コンポーネントの基本的な使い方を確認するためのサンプルです。

## ファイル

| ファイル | 説明 |
|----------|------|
| `sample.tui.yml` | MVP コンポーネント一式（Text, Alert, Input, Checkbox, Radio, Select, Button, Form など）の使用例。最初に開くのに適しています。 |
| `component-style-test.tui.yml` | 各コンポーネントのスタイル・レイアウトの動作確認用。grid レイアウトやネストした Container/Form を含みます。 |

## 使い方

1. `sample.tui.yml` または `component-style-test.tui.yml` を VS Code で開く
2. コマンドパレット（Ctrl+Shift+P）→「TUI: Preview」でプレビューを表示
